# Rock Paper Scissor
* This is a Rock-Paper-Scissor game covered in odin projects's foundation course, to demonstarate & practice the javaScript learnt so far.
* This mini console-only game is made using javaScript, both the inputs & outputs are displayed using alert boxes & also logs outputs to the console

# Why does this exist?
* To practice writing javaScript
* To practice writing clean, maintainable code 

# What did I actually do?
* Took input from user
* Used Math.random() to randomly select between rock, paper, scissor and treat it as computer's choice
* Find out the winner using simple if-else ladder
* Show relevant messages along with scores
* Repeat the above steps for 5 rounds

* I could have improved readability of code
* Could've desinged what & how each function does a bit more carefully
